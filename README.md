# Resume

A responsive, print-friendly personal resume built with **HTML**, **Tailwind CSS**, and accessible best practices. Includes downloadable PDF support and GitHub Pages deployment.

## Features
- Mobile-first responsive design  
- Tailwind CSS utility-first styling  
- Downloadable PDF version  
- Print-optimized layout  
- Links to GitHub and LinkedIn  
- Projects section with live demos

## Tech Stack
- HTML5  
- Tailwind CSS  

## Setup

```bash
# Clone the repo
git clone https://github.com/IntToLong/Resume.git
cd Resume

# If using Tailwind CLI
npx tailwindcss -i ./input.css -o ./output.css --watch
